## 🚀 Projects

### 🥊 UFC Historical Analysis
**Tools:** Excel, Pivot Tables, statistical analysis  
**Focus:** sports analytics, aggregation, regression

Analyzed historical UFC fight data (1993–2021) to answer key business questions about fighter participation, fight outcomes, and performance patterns.

**Key highlights:**

- analyzed 5,700+ fights  
- identified TOP 10 fighters by wins  
- computed KO/TKO finish rates  
- segmented fights by weight class  
- performed correlation & regression (r ≈ 0.80, R² ≈ 0.64)

📂 Project folder: `ufc-historical-analysis`

---

### 🚗 US Car Sales 2019 Analysis
**Tools:** Excel  
**Focus:** price distribution, correlation analysis

Explored the US automotive market using a dataset with 31 variables (price, horsepower, manufacturer, model year).

**Key highlights:**

- price distribution analysis  
- outlier detection  
- manufacturer segmentation  
- hp vs price relationship analysis  

📂 Project folder: `us-car-sales-2019`
