# CodersLab Data Science Bootcamp

This repository documents my learning journey through the **CodersLab Data Scientist Bootcamp**.  
The program covers the full **data science workflow**, from data analysis fundamentals to machine learning and data storytelling.

The course combines theory, practical exercises, and projects designed to simulate real-world analytical work.

---

## Program Overview

The bootcamp focuses on developing practical skills required for **data analysis and data science roles**, including:

- Understanding the **Data Science process**
- Working with **relational databases and SQL**
- Automating analysis with **Python**
- Creating **data visualizations and dashboards**
- Applying **machine learning techniques**
- Communicating insights through **data storytelling**

---

## Course Modules

### 1. Introduction to Data Analysis
- Data Science process
- Framing analytical problems
- Data collection and data cleaning
- Exploratory data analysis
- Basics of statistics
- Linear regression fundamentals
- Introduction to A/B testing
- Data storytelling fundamentals

---

### 2. SQL – Data Analysis
Topics include writing efficient and complex SQL queries and working with relational databases:

- Filtering and formatting datasets
- Subqueries and UNION
- Table relationships
- Joins and join best practices
- Grouping and aggregation
- Window functions
- Working with datetime
- Procedural SQL
- Transactions, loops and cursors

---

### 3. Python – Data Analysis
Python is used to automate analysis and work with external data sources:

- Python fundamentals (functions, lists, tuples, strings)
- Working with files and exceptions
- Using Python libraries
- SQL, JSON and APIs
- Data analysis with **Pandas**
- Data transformation and merging datasets
- Web scraping (HTML parsing with BeautifulSoup)
- Data visualization with Python
- Generating automated reports and documents

---

### 4. Data Visualization
Focus on communicating insights clearly through visual analytics:

- Data visualization theory
- Seaborn and Plotly
- Interactive dashboards with **Dash**
- Graph analysis and network visualization
- Geospatial analysis with **GeoPandas**
- Dashboard design principles
- Data storytelling
- Tableau dashboards

---

### 5. Machine Learning
Introduction to machine learning concepts and practical model building:

**Regression**
- Linear regression
- Polynomial regression
- Decision tree regression
- Model evaluation metrics

**Classification**
- Logistic regression
- K-Nearest Neighbors
- Decision Trees
- Support Vector Machines

**Advanced Models**
- Random Forests
- Boosting and Bagging
- Neural networks
- Natural Language Processing (NLP)

**Unsupervised Learning**
- Clustering
- Dimensionality reduction
- Anomaly detection
